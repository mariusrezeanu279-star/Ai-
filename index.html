About
Ai— Studio — single-file AI toolkit.
Supports Featherless, Venice, Atlas Cloud, OpenAI (via proxy), Anthropic, Hugging Face, GitHub AI Models, Mistral, Groq, OpenRouter, DeepSeek, Google Gemini, Together AI, Pollinations.ai, Cerebras, SambaNova, Hyperbolic, and any OpenAI-compatible API.
Pollinations.ai — 100% free, no API key required. Use it right away.
Cerebras & SambaNova — free-tier accounts available; fast open-source models.
OpenAI requires a CORS proxy (use Universal API tab + your Cloudflare Worker). Hugging Face chat uses the Serverless Inference API.
GitHub AI Models — use a GitHub Personal Access Token (PAT) with Models: Read permission from github.com/settings/tokens.
All data stays in your browser. No tracking, no accounts.
